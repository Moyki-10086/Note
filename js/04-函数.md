### 函数式编程
运行结果，只依赖输入参数，不依赖外部状态

#### 纯函数
没有副作用：相同输入，有相同输出- 
##### 有副作用
例如：watchEffect(vue3)
```js
let sum=10
function add(a){
  return sum + a
}
add(5)
sum=20
add(5)
```
##### 纯函数
例如：filter
```js
function add(a,b){
  return a + b
}
```
